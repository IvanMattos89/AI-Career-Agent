# AI Career Agent

Aplicação desktop em Python para importar, ler e organizar currículos, com base em PySide6, SQLite e módulos de serviços.

## Status atual
- Interface principal com menu lateral
- Navegação entre páginas
- Importação de currículos PDF/DOCX
- Leitura de texto do currículo
- Persistência em SQLite

## Executar localmente
```powershell
python run.py
```
